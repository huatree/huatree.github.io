绘画AI之Midjourney

发表于 2023-03-25 更新于 2024-03-11 分类于 生活滋味 阅读次数： 本文字数： 1.1k 阅读时长 ≈ 1 分钟

加入 Discord 或 Web 上的 Midjourney 社区，数以千计的人在这里合作，通过简短的文字描述创造新的世界、奇妙的角色和独特的图像。

官方文档

关于Midjourney

Midjourney 是一个独立的研究实验室，致力于探索新的思维方式并扩展人类的想象力。

了解更多，详见

快速开始

了解如何使用 Discord 上的 Midjourney Bot 根据简单的文本提示创建自定义图像。

加入Discord

前往 Midjourney.com，选择加入 Beta，或直接前往Midjourney Discord。

在加入 Midjourney Discord 服务器之前，您必须登录 Discord。
在此处了解如何创建一个。

寻找新手频道

在登录后的Discord界面，选择newbies-#左侧边栏中可见的任何频道。

使用 /imagine 命令

使用命令与 Discord 上的 Midjourney Bot 交互。命令用于创建图像、更改默认设置、监视用户信息以及执行其他有用的任务。该/imagine命令从简短的文本描述（称为Prompt）生成一个独特的图像。

  1. 从斜杠命令弹出窗口中键入/imagine prompt:或选择命令。/imagine
  2. 在“提示”字段中输入您要创建的图像的描述。
  3. 单击返回以发送您的消息。

尊重社区准则。社区准则适用于任何使用 Midjourney Bot 的地方。

处理作业

Midjourney Bot 需要大约一分钟的时间来生成四个选项。

生成图像会激活免费的 Midjourney 试用版。试用用户在需要订阅之前可以完成大约 25 个工作。

作业是使用 Midjourney Bot 的任何操作。工作包括使用/imagine命令创建图像网格、放大图像或创建图像变体，全部使用您的免费试用时间。

使用/info命令检查你的快速剩余时间查看您的剩余试用时间。

高档或创造变化

初始图像网格生成完成后，会出现两行按钮：

保存你的图像

单击图像以全尺寸打开它，然后右键单击并选择Save image。在手机上，长按图片，然后点击右上角的下载图标。

所有图片均可立即在midjourney.com/appSign In with Discord上查看。

订阅计划

试用用户大约有 25 个免费工作。工作目前不会过期，但也不会更新。要制作更多图像，请使用/subscribe任何机器人频道中的命令生成指向Discord帐户页面的个人链接。

销售你的绘画指令

如果你有好的绘画指令，怎么去把你的指令卖出去呢？

查找最佳指令，产生更好的结果，节省 API 成本，销售您自己的指令，可通过指令交易市场https://promptbase.com/上进行交易。这就是利用AI创造价值获得收益的一种方式。

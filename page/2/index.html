如何手动加入npm依赖

发表于 2022-04-23 更新于 2022-06-06 分类于 工作点滴 本文字数： 198 阅读时长 ≈ 1 分钟

起因

当遇到一些特定版本的包，刚好网络暂时无法下载，又急需运行调试时，可以借助同事本地的离线包，手动加入 npm 依赖。

解决

手动加入 npm 依赖操作如下：

  1. 检查本地项目所在目录下的 node_modules；如果存在，先删除
  2. 检查 package.json 是否存在需要加入的 npm 依赖；如果是，删除
  3. 执行 npm i 安装依赖
  4. 需要加入的依赖包移入 node_modules 目录
  5. package.json 中加入该 npm 依赖信息

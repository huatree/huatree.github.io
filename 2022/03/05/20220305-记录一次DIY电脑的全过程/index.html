记录一次DIY电脑的全过程

发表于 2022-03-05 更新于 2024-03-11 分类于 生活滋味 阅读次数： 本文字数： 1.1k 阅读时长 ≈ 1 分钟
在这个显卡溢价横流的世界里，DIY一台电脑，抛开价格的约束，单从显卡等级与售价匹配度出发，相信在座的各位都不愿交智商税。

硬件准备

序号 硬件 名称 售价
1 CPU i5-10400(版 U 套装) 1799
2 主板 微星 B560M WIFI 0
3 显卡 蓝宝石 RX6500XT 白金版 1899
4 硬盘 凯侠 RC20 M.2 NVME PCIE3.0 1T 579
5 内存 威刚 DDR4 3200 2*8G 369
6 散热器 利民 AS120 刺灵 99
7 电源 长城 额定 500W HOPE-6000DS(双 8pin) 219
8 机箱 游戏帝国 寂静岭 隔音 109
9 显示器 AOC 27 英寸 AH-IPS 广视角 75Hz HDR Mode Type-C 接口 办公节能 低蓝光不闪 旋转升降电脑显示器 I2790PC 1199
10 键盘 戴尔有线键盘 KB216 黑色 40.92
11 鼠标 牧马人无线鼠标可充电式 磨砂黑(银侧翼) 无声静音 800 毫安大锂电 29.9
12 音箱 小度智能音箱 mini 版 随身蓝牙音箱 迷你音响 65

tips

盒装 CPU 自带散热器的，对于是否有必要再购买散热器还是散片 CPU + 主板 + 散热器组合，自己权衡抉择。

组装

这里，涉及上面硬件准备表格中的 1~8。安装步骤如下：

  1. 打开主板上的 CPU 插槽，对准标志小三角，插入 CPU
  2. 主板上找内存插槽，插入内存条
  3. 主板上找 M.2 硬盘插槽，安插 M.2 硬盘
  4. CPU 上涂上硅脂，散热器安装，插 CPU 风扇线
  5. 拆机箱，安装电源，整理线路
  6. 主板装入机箱，插线，先插入机箱上的线（电源跳线，音频线，USB 线），再插入电源上的线（主板供电接口，CPU 供电接口）
  7. 安装显卡，再插入显卡供电接口线

! 注意

  1. 机箱如果带有 RGB 线，直接找到主板上对应的插槽插入即可。
  2. 这里没有装固态/机械硬盘，所以没提到它怎么插线安装，固态/机械硬盘有一个供电接口（接电源）和数据传输接口（接主板），找到对应位置插入即可。
  3. 机箱除了一个 CPU 扇热器风扇，没有装其他风扇，所以没有给出机箱风扇怎么插线安装。后续如果有的话，直接主板上找对应位置插入即可。
  4. 注意散热器方向，一般跟内存条方向一样，如果只有一个风扇，该风扇需安装到内存条的那一侧。
  5. 机箱还是选大一点的，方面接线。另外，看看机箱规格，支持的主板大小，散热器高度，显卡长度等。

Q&A

1、微星 B560M 主板上 CPU 供电是 8+4 的插槽，需要插满吗？

如果 CPU 不是带 K 的或超频老玩家，一般 8pin 供电足够用了。

2、机箱风扇散热方向反了？

方形风扇，四条边，有一边标有风扇转动方向和出风口方向。散热方向与主板接口方向一致。

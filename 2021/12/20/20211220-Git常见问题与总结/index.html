11 设置默认分支

一般不推荐全局设置默认分支，可根据项目需要设置默认分支。如 hexo + github 搭建的静态博客，静态资源所在的分支名一般命名为gh-pages，设置其为默认分支。

12 取消暂存区的更改

更改的文件不会还原到未更改的状态，只是把已更改的文件从暂存区移到了工作区，这个移到，美其名曰撤销。

13 恢复误删分支

不用担心是谁创建的恢复分支，因为你并没有 commit，只是依赖指定的 commit id 创建分支，不会有个人信息记录。
